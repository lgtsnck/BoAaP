# Основы алгоритмизации и программирования

Данный репозиторий содержит цикл лабораторных работ, выполненных в рамках учебной дисциплины **"Основы алгоритмизации и программирования"**.

## 🛠 Технологический стек
* **Язык программирования:** C
* **Стандарт языка:** C23
* **Среда разработки (IDE):** CLion 2025.3.1.1
* **Система сборки:** CMake
* **ОС:** Windows 10 22H2 и выше

## 📂 Структура репозитория
Проект организован по модульному принципу. Каждая директория `lab*` представляет собой отдельную законченную работу:
* **Графика WinAPI:** Рисование геометрических примитивов и сложных логотипов.
* **Алгоритмы на массивах:** Обработка одномерных и двумерных массивов, динамическое выделение памяти.
* **Работа с файлами:** Чтение данных из текстовых файлов и генерация HTML-отчета.
* **Структуры данных:** Работа со статической базой данных объектов и бинарными файлами.
* **Собственные библиотеки:** Реализация аналогов функций стандартной библиотеки `string.h`.

## 🚀 Инструкция по запуску
Для корректной проверки и запуска работ выполните следующие действия:

1. Склонируйте проект на локальный компьютер:
   ```bash
   git clone https://github.com/lgtsnck/BoAaP.git
2. Откройте необходимый каталог lab* как отдельный проект в вашей IDE (**крайне рекомендуется CLion**).
3. Сборка каждой лабораторной работы осуществляется автоматически на основе существующего в папке файла CMakeLists.txt.
4. Запустите выполнение проекта через запуск функции main().
